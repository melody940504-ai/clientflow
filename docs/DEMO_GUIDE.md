# Lumaire Demo Guide

This walkthrough takes about two minutes and covers both sides of the product without changing the shared data.

## Studio View

1. Open [Lumaire](https://clientflow-q250.onrender.com/) and choose **Explore studio demo**.
2. Scan the project status summary and the seeded Northstar Creative workspace.
3. Open **Launch Reels Package** to see its versions, attachments area, review controls, and activity timeline.
4. Return to the dashboard and open **Analytics** from the menu.
5. Open **Settings** to preview workspace branding and color controls.
6. Use the bell beside the menu to inspect project notifications.

The shared studio account is read-only. Creation, upload, branding, delivery, and review actions are intentionally disabled.

## Client View

1. Log out, return to the landing page, and choose **Open client view**.
2. Open an assigned Acme Studio project.
3. Compare versions and review the comment, revision, and approval controls.
4. Resize the browser or open the page on a phone to see the mobile client workflow.

The client demo is also read-only, so all seeded projects remain stable for the next visitor.

## Full Workflow in a Private Account

For a writable test, create a separate owner account and complete this path:

1. Verify the email and configure workspace branding.
2. Create a client and project.
3. Upload a version or provide a hosted video URL.
4. Open the generated public review link in a private browser window.
5. Leave feedback, request a revision, or approve the version.
6. Return to the owner dashboard to inspect the notification, timeline, status, and analytics update.
